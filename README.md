# 物资管理系统

## 项目简介
物资管理系统基于java Web/SSM框架/JPS技术，Mysql数据.数据库包含五张表，物资管理系统对五张数据库表进行增删改查操作。适合作为课程设计、大作业、毕业设计等

Java/SSM框架/JSP/Mysql数据库
